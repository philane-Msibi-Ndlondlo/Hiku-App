# Progress Tracker App - Hiku

This is a web and desktop app that tracks user's progress in terms of tasks and subtasks.

# Features

1. Login
2. Registration
3. Add a Project
4. Add Project Tasks
5. Progress Tracking: Not Started, In Progress and Completed.
___

# Tech Stack

1. React
2. SASS
3. Electron JS
4. LocalStorage
5. Express JS
6. TailWind CSS
